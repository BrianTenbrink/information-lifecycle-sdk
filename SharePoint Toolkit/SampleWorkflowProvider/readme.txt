
In order to deploy the custom workflow provider, perform the following steps:

1. Copy the compiled assembly to the following location on each SharePoint Web Front End:

	SharePoint 2013
	C:\Program Files\Common Files\microsoft shared\Web Server Extensions\15\TEMPLATE\LAYOUTS\RecordsManagement\Providers\Workflow

	SharePoint 2010
	C:\Program Files\Common Files\microsoft shared\Web Server Extensions\14\TEMPLATE\LAYOUTS\RecordsManagement\Providers\Workflow

2. IIS Reset
